Portfolio

  • Garage door web application with live updates based on state of the garage, ability to toggle garage door, toggle light, and update advanced settings such as automatically close and automattically shut light off. Commnicates with two Particle Photons- one to simulate the garage door and one to act as a garage door remote controller. Remote controller has OLED screen that also updates based on the garage's current state.
  • Technologies used: Particle's Photon, Javascript, C++, Particle's Cloud API and Javascript API
  • Built by: Sam Margolis and Soraya Moss
  • Online chat room service that allows users to create a nickname, create public or private chatrooms, or direct message users. Creators of chat rooms have the ability to kick out or ban specific users from using the room. All forms of profanity are censored within the chat room and if a user uses profanity too often they are kicked out of the chatroom. SQL database keeps track of quantity of swear words uses by each user and warns users if they have been using too much profanity. Users may also report other users to the creator of the chat room.
  • Technologies used: Javascript, Node.js, Socket.io, Amazon Web Services
  • Built by: Sam Margolis and Soraya Moss
  • Simplified version of the classic video game, Gorilla's. Users can enter a desired player name and a map of a variety of building heights and locations for the players is generated randomly. Players proceed to enter in desired velocities and angles for their projectiles to shoot at the opposing player. Explosion simulations follow upon impact of the projectile. The players can also specify how many games they would like to play, `and the winner is congratulated upon completion of the game.
  • Technologies used: Java
  • Built by: Sam Margolis
  • Javascript calendar that allows users to add, delete, and update events dynamically. The calendar uses AJAX with PHP server-side scripts that query a MySQL database to save and retrieve information, including user accounts and events. The calendar notifies the user if there are any events on the current day. The user also has the ability to invite other users to their own events, which are propogated in and invite list and displayed in a JQuery pop-up dialog box.
  • Technologies used: Javascript, PHP, AJAX, Amazon Web Services
  • Built by: Sam Margolis and Soraya Moss
